# Membership Operators


## in

```python
name = "GeoNext"
print("G" in name)
```

```python
lst1 = [1, 2, 3]
```

```python
print(5 in lst1)
```

## not in

```python
name2 = "Apple"
```

```python
print("N" not in name2)
```

```python
lst2 = [5, 6, 7, 10]
```

```python
print(10 not in lst2)
```
