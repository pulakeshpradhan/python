# Comparison Operators

```python
a = 10 # a < b
b = 15
```

## Equal to (==)

```python
a == b
```

## Not equal to (!=)

```python
a != b
```

## Greater than (>)

```python
a > b
```

## Less than (<)

```python
a < b
```

## Greater than or equal to (>=)

```python
a >= b
```

## Less than or equal to (>=)

```python
a <= b
```
