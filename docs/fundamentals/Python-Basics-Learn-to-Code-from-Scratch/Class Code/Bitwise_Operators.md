# Bitwise Operators


## & (Bitwise AND)

```python
a = 20
b = 8
```

```python
print(a & b)
```

```python
x = 7
y = 5
```

```python
print(x & y)
```

## | (Bitwise OR)

```python
print(a | b)
```

## ^ (Bitwise XOR)

```python
print(a ^ b)
```

## ~ (Bitwise NOT)

```python
print(~a)
```

```python
c = 28
print(~c)
```

## << (Bitwise left shift)

```python
print(a << 2)
```

## >> (Bitwise right shift)

```python
print(a >> 2)
```
